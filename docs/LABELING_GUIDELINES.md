# ラベル付け基準

## 1. 目的

本基準は、`prone` と `non_prone` のラベル境界、および学習利用可否判定を統一するための基準である。

## 2. 姿勢ラベル定義

### 2.1 `prone`

以下をすべて満たすとき `label=1` とする。

- 被写体が腹側を下にしている
- 主体姿勢がうつ伏せである
- 一時的な動作途中ではなく、撮影時点の姿勢としてうつ伏せと判断できる

### 2.2 `non_prone`

以下を満たすとき `label=0` とする。

- 仰向け、横向き、抱っこ、座位など、主体姿勢がうつ伏せではない
- 撮影時点でうつ伏せと明確に区別できる

## 3. 学習利用可否

### 3.1 学習利用可

以下を満たすとき `is_usable_for_training=1` とする。

- 姿勢ラベルに迷いがない
- 被写体が主対象として十分写っている
- ブレ、露光、遮蔽が学習に支障ない
- 被写体が単独で写っている
- `96 x 96` 入力へ縮小しても体幹姿勢が判別できる

### 3.2 学習利用不可

以下のいずれかに該当する場合 `is_usable_for_training=0` とする。

- 姿勢が曖昧
- 顔または体の見え方が極端に不足
- 被写体が不在
- 被写体が複数
- 画質が著しく悪い
- ラベル付け判断に自信がない
- 被写体が小さすぎて縮小後に姿勢特徴が失われる

## 4. `exclude_reason` 判定基準

### 4.1 `ambiguous_pose`

- うつ伏せか非うつ伏せか即断できない
- 寝返り途中
- 体幹の向きが隠れている

### 4.2 `face_not_visible`

- 顔情報が主題ではないが、姿勢確認に必要な見え方が不足している
- 顔や頭部が完全に隠れて姿勢監査が困難

### 4.3 `subject_missing`

- 被写体が画面にいない
- 被写体が極端に小さく実質不在

### 4.4 `multiple_subjects`

- 2 人以上の乳幼児が写る
- 大人が大きく写り込み、主被写体判定が不安定

### 4.5 `motion_blur`

- 体幹や顔が流れて姿勢確認が難しい

### 4.6 `poor_lighting`

- 暗すぎる
- 白飛びが大きい
- 強い逆光で姿勢判断が困難

### 4.7 `annotation_error`

- 入力ミスが判明した
- 保存後レビューでラベルが誤りと判明した

### 4.8 `other`

- 上記分類に当てはまらないが学習利用不可

## 5. 優先順位

複数理由がある場合は、最も根本的な理由を優先する。

優先順:

1. `subject_missing`
2. `multiple_subjects`
3. `ambiguous_pose`
4. `motion_blur`
5. `poor_lighting`
6. `face_not_visible`
7. `annotation_error`
8. `other`

## 6. 迷ったときのルール

- 迷ったら `is_usable_for_training=0`
- `non_prone` に寄せて無理に確定しない
- 再レビュー可能なよう画像は残す

## 7. レビュー運用

- 初回ラベル付け者とレビュー者を分けられる場合は分ける
- レビューで誤りが見つかったら `annotation_error` を使って管理する
- ラベル修正履歴が必要なら PC 側で監査ログを持つ
